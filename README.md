# README #

WMS application

### What is this repository for? ###

* ios
* ipad
* Version 0.0.1

### Setting up the development environment ###

* node: 10.16.3
* react-native: 0.63.3
* pods: 1.11.3

## Start application:

* nvm install 10.16.3
* nvm use 10.16.3
* rm -rf node_modules
* npm install
* cd ios
* pod install
* react-native run-ios --simulator="iPad mini (6th generation)"

## OR ###

* rm package-lock.json
